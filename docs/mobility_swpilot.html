Self-Driving Mobility Kit Build

동키카(Donkey Car) 키트와 컴퓨터 비전을 활용하여 자율주행 알고리즘을 구현하는 10일 실무 과정

이 과정은 대구디지털혁신진흥원(DIP) 지원 하에, 소형 모빌리티 플랫폼 하드웨어 조립부터 임베디드 OS 환경 구축, OpenCV 기반 컴퓨터 비전 처리, 그리고 딥러닝(CNN) 주행 학습 모델의 실제 구동까지 경험하는 융합형 프로젝트입니다.
