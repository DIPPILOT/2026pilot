# 대구디지털혁신진흥원(DIP) SW/AI 교육과정 대학생 맞춤형 LMS UI/UX 디자인 제안서

본 제안서는 대구디지털혁신진흥원(DIP)에서 주관하는 SW/AI/ICT 교육과정에 참여하는 대학생들의 학습 몰입도를 높이고, 실무 중심의 성장 과정을 효과적으로 관리할 수 있는 차세대 **LMS(Learning Management System) 'DIP-LINK'**의 UI/UX 디자인 컨셉을 제안합니다.

---

## 1. 디자인 컨셉 및 방향성 (Design Concept)

> [!NOTE]  
> **핵심 슬로건: "Learn, Play, and Port-in"**  
> 딱딱하고 지루한 학사 관리 도구에서 탈피하여, 대학생들이 실질적인 IT 역량을 기르고 포트폴리오로 연계할 수 있는 '성장 플랫폼'을 지향합니다.

### 🌟 3대 디자인 필러(Pillars)
```
┌────────────────────────┐      ┌────────────────────────┐      ┌────────────────────────┐
│  1. Tech-Friendly UI   │      │  2. Gamified Tracker   │      │ 3. Portfolio-Ready UX  │
├────────────────────────┤      ├────────────────────────┤      ├────────────────────────┤
│ 개발자/디자이너 감성의 │      │ 게이미피케이션 요소를  │      │ 일일 학습 결과가 자동으로│
│ 다크 모드, 코드 친화적  │      │ 도입한 잔디 심기 및   │      │ 포트폴리오로 변환 및    │
│ 레이아웃 및 폰트 적용  │      │ 챌린지 성취율 시각화    │      │ 기업 매칭으로 연계되는 UX│
└────────────────────────┘      └────────────────────────┘      └────────────────────────┘
```

## 2. 비주얼 가이드라인 (Visual Identity)

대학생들의 선호도가 높은 테크니컬하고 감성적인 **글래스모피즘(Glassmorphism)**과 **네오 브루탈리즘(Neo-Brutalist)** 요소를 혼합하여 직관성과 트렌디함을 모두 확보합니다.

### 🎨 컬러 팔레트 (Color Palette)

| 구분 | 색상 코드 | 프리뷰 | 활용처 |
| :--- | :--- | :--- | :--- |
| **Primary (DIP Tech Blue)** | `#1E293B` | `██████` | 메인 네비게이션, 헤더, 배경 카드 (다크모드용) |
| **Brand (DIP Indigo)** | `#3B82F6` | `██████` | 메인 브랜드 강조색, 버튼 활성화, 진척도 바 |
| **Point (Digital Teal)** | `#14B8A6` | `██████` | 실습 성공 알림, 과제 완료 표시, 챗봇 및 AI 비서 |
| **Accent (Energetic Coral)**| `#F43F5E` | `██████` | 미완료 과제, 기한 임박 알림, 중요 오류 메세지 |
| **Background (Light Mode)**| `#F8FAFC` | `██████` | 전체 대시보드 배경, 카드 컴포넌트의 은은한 그림자 |

### ✍️ 타이포그래피 (Typography)
*   **대표 폰트**: `Pretendard JP/KR` (가독성이 극대화된 기본 산세리프) + `JetBrains Mono` (코드 블록 및 개발 지표용 고정폭 폰트)
*   **크기 규칙**:
    *   `Display Title`: 32px (학습 대시보드 상단 환영 메시지)
    *   `Section Header`: 20px (각 카드 섹션 타이틀)
    *   `Body Text`: 14px/15px (공지사항 및 학습 요약)

## 4. 특화 UI/UX 디자인 제안 (Core Feature Design)

### ① 개발자 감성의 '학습 잔디(GitHub Style)' 심기 컴포넌트
*   **컨셉**: GitHub의 커밋 기록(Contribution Graph)에서 영감을 얻어, 매일 동영상 강의 시청, 과제 제출, 퀴즈 풀이 등을 수행할 때마다 초록색 잔디가 심어지도록 디자인합니다.
*   **대학생 유인책**: 친구들과 잔디 심기 랭킹을 쉐어할 수 있는 SNS 카드 공유 기능 제공.

```
[ 잔디 밭 시각화 예시 ]
 2026.06 (Day 1 ~ 30)
 ◼ ◼ ◼ 🟩 🟩 ◼ ◼ 
 ◼ 🟩 🟩 🟩 🟩 🟩 ◼ 
 🟩 🟩 🟩 🟩 🟩 🟩 🟩  <- 채워지는 재미를 통한 일일 학습 동기 부여!
```

### ② 실시간 1:1 AI 튜터 챗봇 & 버그 리포터 (오른쪽 하단 플로팅 버튼)
*   **컨셉**: DIP 교육 특성상 코딩 과제가 많으므로, 에러 발생 시 즉각적으로 코드 스니펫을 붙여넣고 피드백을 받을 수 있는 플로팅 위젯.
*   **비주얼**: 사이버틱한 테일(Teal) 컬러의 네온 글로우(Neon Glow) 효과를 준 원형 아이콘. 클릭 시 반투명 슬라이드 아웃(Slide-out) 패널 작동.

### ③ '마일스톤 로드맵' 내비게이션
*   **컨셉**: 수료까지의 전체 과정을 RPG 게임의 퀘스트 맵처럼 시각화.
*   **디자인**: 
    - 완료된 세션: 컬러 플래그와 통과(Pass) 마크.
    - 진행 중인 세션: 깜빡이는 스팟 애니메이션.
    - 미오픈 세션: 반투명 잠금(Lock) 아이콘.

## 5. UI 컴포넌트 마크업 예시 (CSS 가이드 포함)

대시보드 내 **학습 진척도 카드**의 UI 코드를 구현하기 위한 디자인 시스템 예시입니다.

```css
/* Glassmorphism Card Style */
.lms-card {
  background: rgba(255, 255, 255, 0.7);
  backdrop-filter: blur(12px);
  border: 1px solid rgba(255, 255, 255, 0.3);
  border-radius: 16px;
  padding: 24px;
  box-shadow: 0 8px 32px 0 rgba(31, 38, 135, 0.05);
  transition: all 0.3s ease;
}

.lms-card:hover {
  transform: translateY(-4px);
  box-shadow: 0 12px 40px 0 rgba(31, 38, 135, 0.1);
  border-color: rgba(59, 130, 246, 0.5); /* DIP Blue hover 효과 */
}

/* Gamified Progress Bar */
.progress-bar-container {
  width: 100%;
  height: 8px;
  background-color: #E2E8F0;
  border-radius: 9999px;
  overflow: hidden;
}

.progress-bar-fill {
  height: 100%;
  background: linear-gradient(90deg, #3B82F6 0%, #14B8A6 100%);
  border-radius: 9999px;
  transition: width 0.5s cubic-bezier(0.4, 0, 0.2, 1);
}
```

## 6. 결론 및 기대 효과

*   **몰입도 극대화**: 대학생들에게 익숙한 Github 디자인 및 게이미피케이션 인터페이스로 자연스러운 자율 학습을 유도합니다.
*   **원스톱 취업 연계**: DIP가 지역 인재 채용을 활성화하는 기관인 만큼, LMS에 누적된 실습 코드와 과제 성취도가 실시간 포트폴리오로 바로 출력되어 대구 지역 IT 기업 채용 매칭 플랫폼으로 다이렉트 연동되는 UX를 기대할 수 있습니다.
